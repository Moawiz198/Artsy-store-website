# Artsy Donut 🍩

A premium, high-end art portfolio and e-commerce platform for exquisite calligraphy and custom art pieces.

## 🎨 Features
- **Curated Art Gallery**: Showcasing a blend of traditional and modern calligraphy.
- **Custom Commissions**: Order personalized portraits, Surahs, or crochet art.
- **Premium UI**: Dark-themed, atmospheric design with smooth animations.
- **Admin Dashboard**: Effortlessly manage products, orders, and inquiries.

## 📍 Location
Based in **Rahim Yar Khan, Punjab, Pakistan**.

## 🚀 Tech Stack
- **Frontend**: React + Vite + Vanilla CSS
- **Backend**: Supabase (Database & Auth)
- **State Management**: React Hooks
- **Animations**: Framer Motion / Custom CSS

## 🛠️ Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
